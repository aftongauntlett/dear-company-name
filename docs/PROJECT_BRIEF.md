# Project Brief

## Client Snapshot

- **Name:** Dear Company
- **Project type:** Personal application microsite (job application)
- **Industry:** Frontend development
- **Mission:** Replace a video submission with a polished, personality-forward personal microsite I can target at specific companies.
- **Primary audience:** Hiring managers and engineering leads.

## Success Criteria

1. A hiring manager can quickly understand my skills, process, and personality from a single scroll.
2. Navigation and CTA flow follow page order top-to-bottom, with nav labels matching section intent and in-page links landing on the correct sections.
3. All sections pass WCAG 2.2 AA, typecheck, lint, tests, and production build.
4. If the GitHub-backed explorer is unavailable, the section gracefully shows a temporary issue state without breaking the rest of the page.
